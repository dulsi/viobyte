About

In the tiny digital world of electrons, the Viobyte virus collects data. It must evade the Generic Host Offensive Software Terminators (GHOST).

Viobyte is a free clone of the arcade game "Pac-Man". It was developed for the TinyScreen Video Game Kit. It is not intended to be a direct clone but instead evoke the feel of the original game.
